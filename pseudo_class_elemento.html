Trabalhando com Pseudo-classe

São usadas para definir um elemento com um estado espacial

Serve para:

  • Estilizar um elemento quando um usuário passa o mouse sobre ele.
  • Estilizar o link visitando e o não visitado de forma diferente.
  • Estilizar um elemento quando ele recebe o foco.

PSEUDO-CLASSE

Pseudo-elemento

Um pseudo-elemento é usado para estilizar partes especiificas de um elemento.

Por exemplo:

  • Estilize a primeira letra ou linha de um elemento.
  • Insira conteúdo antes ou depois do conteúdo de um elemento.

    • Estilizando a primeira linha

    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Odit officiis asperiores provident corporis! Eum est aliquid harum necessitatibus ratione dolorem perspiciatis voluptas ea voluptatem placeat voluptatum officiis, sapiente veniam labore.

NOTA:O ::first-line só pode ser aplicado a um elemento de nível de bloco.

As seguintes propriedades se explicam ao ::first-line:

  • Propriedade da fonte
  • Propriedade de cor
  • Propriedade de fundo
  • Espaçamento entre as palavras
  • Espaçamento entre as letras
  • Descoração do texto
  • Alinhamento vertical
  • Transformação de texto
  • Altura da linha
