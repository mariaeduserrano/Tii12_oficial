Trabalhando com ID e Classe

Seletores ID

É usado um seletor de ID em um elemento HTML para deixar ele único e específico.

Deve ser usado uma única vez na página.

Exemplo

USANDO UM ID

Seletor de classe

O seletor de classe seleciona elemento com um atributo de classe específico.

Pode ser usado mais se uma vez.

Exemplo

USANDO CLASSE

USANDO CLASSE

USANDO CLASSE

OUTRO TEXTO

USANDO CLASSE
